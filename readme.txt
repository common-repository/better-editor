== Installation ==

* FROM WP ADMIN: Install from the WordPress.org respository by going to **Plugins > Add New** and search for **Better Editor**.  Then click on **Install** & **Activate**.

* BY UPLOADING: Upload and extract the zip file and just drop the contents in the **wp-content/plugins/** directory of your WordPress installation and then activate the Plugin from the **Plugins** page.

== Frequently Asked Questions ==

= How to edit posts? =

Below each post/page there is a "Edit" button added. When you click on this button you will be able to edit the text area for the selected post.

= Can I change the toolbar design? =

Yes, go to **Settings > Better Editor** and choose one of the 7 predefined themes.
